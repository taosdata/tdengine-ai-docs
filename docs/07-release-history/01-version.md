# 版本列表

TDengine IDP 版本号采用三位版本号，由 `MAJOR.MINOR.PATCH` 三部分组成，中间由`.`分隔，具体定义如下：
- `Major`：产品有重大重构，不能直接升级，如需升级，请联系 TDengine 客户支持团队
- `Major`：有重大新特性，升级后不可回退
- `Patch`：有新特性或缺陷修复，升级后可以回退
